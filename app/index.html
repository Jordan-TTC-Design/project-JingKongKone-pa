石門金剛宮

台灣最北端宗教博物館

相關報導

金剛宮是北台灣宮廟規模數一數二的佛道教聖地，宮內的藏寶光明樓還有一大特點，那就是殊聖無比的世界第一大臥佛，位於一樓殿內的臥佛身長59.8台尺，規模之大令人讚嘆不已，慈眉善目的佛像全身金光閃閃，顯得富貴莊嚴、宏偉與慈悲兼具，是到金剛宮不得不探訪的重點之一

關於我們

金剛宮創立於民國七十五年 ( 西元1986年 ) 是以提倡社教功能，興辦各公益活動，關懷與愛心扶助弱勢團體，純淨宗教文化，發揮神佛博大精深，普愛世人之精神為宗旨。 金剛宮以四面佛為主祀，解決善信現世之業障祁願，同時普納儒釋道三教，萬神匯光、千佛自在、廣陳供養以淨化人心，建立「以佛治心、以道淨意、以儒戒行」的新宗教觀，不惟敬祀於天神，也順奉於父母，不惟專修於宗敎聖典，也不怠於人事族群， 此即為石門金剛宮出世入世兼具創宮主旨精神。 金剛宮創立於民國七十五年 ( 西元1986年 ) 是以提倡社教功能，興辦各公益活動，關懷與愛心扶助弱勢團體，純淨宗教文化，發揮神佛博大精深，普愛世人之精神為宗旨。 金剛宮以四面佛為主祀，解決善信現世之業障祁願，同時普納儒釋道三教，萬神匯光、千佛自在、廣陳供養以淨化人心，建立「以佛治心、以道淨意、以儒戒行」的新宗教觀，不惟敬祀於天神，也順奉於父母，不惟專修於宗敎聖典，也不怠於人事族群，此即為石門金剛宮出世入世兼具創宮主旨精神。
